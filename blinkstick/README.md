## Blinker

### Input ports: 
* color: any

* delay: any

* repeats: any

### Output ports:

## Color setter

### Description:
Sets an RGB color on the specified LED.

### Input ports: 
* color: any

    type: [number, number, number]


* index: any

    type: number


### Output ports: 
* done: any

* bounced: any
